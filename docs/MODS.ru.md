# Список модов

Эта мультиплеерная сборка основана на `Calamity Mod`.

Ниже моды сгруппированы и отсортированы по влиянию на геймплей: от крупных контентных изменений к более легким utility/QoL-модам.

## Основные контентные моды

| Мод | Workshop ID | Описание |
| --- | --- | --- |
| Calamity Mod | `2824688072` | Главный overhaul-мод. Добавляет боссов, биомы, экипировку, изменения прогрессии и основную часть контента сборки. |
| Calamity Entropy | `3316697363` | Крупное дополнение для Calamity с новым оружием, механиками и поздним контентом. |
| AlchemistNPC Lite | `2599842771` | Добавляет полезных NPC, магазины и поддержку прогрессии без лишней тяжести полной версии. |
| Calamity's Vanities | `2824688804` | Добавляет vanity-предметы, питомцев и косметические элементы, связанные с Calamity. |

## Обязательные зависимости

| Мод | Workshop ID | Описание |
| --- | --- | --- |
| Calamity Mod Music | `2824688266` | Пак музыки, необходимый для Calamity Mod. |
| InnoVault | `3338554501` | Библиотека, требуемая для Calamity Entropy. |
| absoluteAquarian Utilities (SerousCommonLib) | `2908170107` | Общая библиотека, требуемая для Magic Storage. |

## Крупные QoL-моды

| Мод | Workshop ID | Описание |
| --- | --- | --- |
| Magic Storage | `2563309347` | Общая система хранения и крафта. Один из самых полезных QoL-модов для мультиплеера. |
| Boss Checklist | `2669644269` | Показывает прогрессию по боссам, нужные предметы призыва и порядок прохождения. |

## Небольшие QoL-моды и локализация

| Мод | Workshop ID | Описание |
| --- | --- | --- |
| Project tRU | `2831360228` | Русская локализация и связанная текстовая поддержка для модифицированной Terraria. |

## Необязательный клиентский мод

| Мод | Workshop ID | Описание |
| --- | --- | --- |
| Calamity Mod Extra Music | `2835214226` | Дополнительный музыкальный пакет для игроков. На сервере не нужен. |

## Примечание

Этот файл теперь служит только справочным списком примерной сборки.

Скрипт установки не скачивает моды. Поместите свои экспортированные `.tmod` файлы в `server_data/Mods`.
