# Terraria tModLoader Server for Ubuntu

Язык:

- English: [../README.md](../README.md)
- Russian: [README.ru.md](./README.ru.md)

Простой мультиплеерный сервер `Terraria` на Ubuntu с использованием `tModLoader`.

Эта сборка основана на `Calamity Mod` и небольшом наборе дополнительных модов для игры по сети.

Список модов:

- English: [MODS.md](./MODS.md)
- Russian: [MODS.ru.md](./MODS.ru.md)

## Что есть в репозитории

- shell-скрипты для установки и запуска
- шаблоны конфигурации
- ID модов для серверной сборки

В репозитории нет файлов Terraria, `tModLoader`, модов или исходного кода игры.

## Требования

- Ubuntu `22.04` или новее
- `tModLoader`
- открытый порт `7777` на роутере или фаерволе, если игроки будут подключаться через интернет

## Требования к серверу

- CPU: минимум современный 2-ядерный процессор
- RAM: минимум `4 GB`, рекомендуется `8 GB` для более стабильного и плавного геймплея
- Диск: минимум `10 GB` свободного места для `tModLoader`, модов и файлов мира

## Быстрый старт

1. Скопируйте репозиторий на Ubuntu-сервер.
2. Создайте локальный конфиг:

```bash
cp .env.example .env
```

3. Отредактируйте `.env`.
4. Сделайте скрипты исполняемыми:

```bash
chmod +x setup.sh start-server.sh
```

5. Запустите установку:

```bash
./setup.sh
```

6. Запустите сервер:

```bash
./start-server.sh
```

## Конфигурация

Основные параметры в `.env`:

```dotenv
SERVER_NAME=My tModLoader Server
WORLD_NAME=modded-world
WORLD_SEED=
WORLD_SIZE=2
WORLD_DIFFICULTY=0
MAX_PLAYERS=8
PORT=7777
PASSWORD=
MOTD=Welcome to the modded server
```

Полезные значения:

- `WORLD_SIZE`: `1` маленький, `2` средний, `3` большой
- `WORLD_DIFFICULTY`: `0` Classic, `1` Expert, `2` Master, `3` Journey

## Seed мира

`WORLD_SEED` необязателен. Если оставить его пустым, Terraria создаст случайный мир.

## Моды

Сервер использует мультиплеерную сборку на базе Calamity из [MODS.ru.md](./MODS.ru.md).

Примечания:

- `Calamity Mod Extra Music` является клиентским модом и не устанавливается на сервер

Во время установки скрипт:

- подготавливает список модов для сервера
- пытается скачать указанные Workshop-моды
- копирует загруженные `.tmod` файлы в папку модов сервера

Игрокам все равно нужен `tModLoader` на их ПК. Недостающие обязательные моды обычно можно скачать при подключении к серверу.

## Файлы

- `.env.example`: шаблон локальных настроек
- `setup.sh`: установка зависимостей, загрузка `tModLoader`, подготовка модов
- `start-server.sh`: создание `serverconfig.txt` и запуск сервера
- `docs/MODS.md`: список модов на английском
- `docs/MODS.ru.md`: список модов на русском
