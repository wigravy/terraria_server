# Mod List

This multiplayer pack is built around `Calamity Mod`.

Mods are grouped below and ordered by gameplay impact, from major content changes to lighter utility additions.

## Main content mods

| Mod | Workshop ID | Description |
| --- | --- | --- |
| Calamity Mod | `2824688072` | The main overhaul mod. Adds bosses, biomes, gear, progression changes, and most of the pack's gameplay. |
| Calamity Entropy | `3316697363` | Large Calamity addon with extra weapons, mechanics, and late-game content. |
| AlchemistNPC Lite | `2599842771` | Adds helpful NPCs, shops, and progression support without being as heavy as the full version. |
| Calamity's Vanities | `2824688804` | Adds vanity items, pets, and themed extras tied to Calamity. |

## Core dependencies

| Mod | Workshop ID | Description |
| --- | --- | --- |
| Calamity Mod Music | `2824688266` | Music package required by Calamity Mod. |
| InnoVault | `3338554501` | Library required by Calamity Entropy. |
| absoluteAquarian Utilities (SerousCommonLib) | `2908170107` | Shared library required by Magic Storage. |

## Major quality-of-life mods

| Mod | Workshop ID | Description |
| --- | --- | --- |
| Magic Storage | `2563309347` | Centralized storage and crafting system. One of the most useful multiplayer QoL mods. |
| Boss Checklist | `2669644269` | Tracks boss progression, summon items, and progression order. |

## Minor quality-of-life and localization

| Mod | Workshop ID | Description |
| --- | --- | --- |
| Project tRU | `2831360228` | Russian localization and related text support for modded Terraria. |

## Client-only optional mod

| Mod | Workshop ID | Description |
| --- | --- | --- |
| Calamity Mod Extra Music | `2835214226` | Optional extra music pack for players. Not needed on the server. |

## Notes

This file is now only a reference list for a sample mod pack.

The setup script does not download mods. Put your own exported `.tmod` files in `server_data/Mods`.
